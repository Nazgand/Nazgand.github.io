TBA Cheat client

TBAdventure (TBA for short) was a flash game made by Attilio Infante. Here is the last snapshot of the highscores. My account was Nazgand Description at TBAdventure Wiki. I volunteered as a security consultant on the supposedly secret TBA admin forum. I used javascript to send people's login details to my website, which logged the data. This happened first in the TBA forums until Attilio Infante stopped allowing javascript in the TBA forums, after which I replicated the bug with javascript injection for anyone who viewed my website. The TBA cheat client source code is lost.

My TBA cheat client could:
  • Teleport to any map
  • Use a modular arithmetic error to add a total of 2^32=4294967296 stat points to the combat stats.
  • Create gold through a trading glitch.
  • Repeatedly teleport to a boss (which was not yet available to regular players), kill it, and loot it. A server-wide message appeared every time the boss was killed, which happened multiple times per second.
  • Auto-loot bones and Auto-bury bones. My account(Nazgand) was the best burier in the game.
